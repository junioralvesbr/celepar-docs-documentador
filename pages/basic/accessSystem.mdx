# Como ter acesso ao Documentador ?
**O acesso ao documentador é controlado por 3 grandes sistemas:**

<Cards style={{alignItems: "center"}}>
  <img src="/central-seguranca.png" />
  <img src="/logo-sentinela.jpg" />
  <img src="/logo-scriba.png" />
</Cards>
<br />
<br />

*Para ter acesso, basta seguir os seguintes passos:*

<Steps>
### 🔑 Central de Segurança
É o sistema que autentica todos os usuarios que utilizam os sistemas da **Celepar**

O usuario precisa fazer o seu cadastro na [Central de Segurança](https://cidadao-cs.identidadedigital.pr.gov.br/centralcidadao/publico/pages/usuario/autoCadastro/autoCadastro.jsf)

<Callout emoji="💡">
  **Importante!!** É recomendado que o próprio usuario faça o seu cadastro na *Central*, pois as informações requeridas são de cunho pessoal.
</Callout>

### 🔑 Sentinela
É o sistema que gerencia o acesso dos usuarios aos sistemas da *Celepar*.

Após o cadastro na *central de segurança*, algum administrador do sistema deverá liberar o acesso do usuario ao sistema Documentador **(RCD)**

### 🔑 ScribaPinhão
É o sistema que gerencia os grupos de usuarios que terão acesso as pastas e documentos do repositório no **Documentador**.

O administrador do sistema deverá adicionar o usuario ao grupo especifico que tem acesso ao repositório

### 🔑 Documentador
O usuario já será capaz de fazer login no *Documentador*

![Documentador](/documentador.png)
</Steps>

<Callout emoji="⚠️">
  **Artenção!!** Caso não haja um **administrador** para fazer as liberações de acesso no **Sentinela** ou no **ScribaPinhão**, você poderá abrir chamado na Celepar através do seu atendedor.
</Callout>
